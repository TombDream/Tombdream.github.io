用 Astro 构建极简个人博客

2026 年 5 月 22 日 · 5 分钟阅读

Astro 是一款专注于内容的静态站点生成器。它的核心理念是零 JS 默认输出——只有当你真正需要交互时，才会发送 JavaScript。对于博客这种以内容为主的场景，这是最佳选择。

为什么选择 Astro？

在评估了多个静态站点生成器之后，我最终选择了 Astro，原因如下：

  • 性能极佳：默认零 JavaScript，首屏加载速度极快
  • 内容优先：对 Markdown / MDX 支持非常友好
  • 框架无关：可以用 React、Vue、Svelte 组件，但只在需要时加载
  • 开发体验好：HMR 速度快，配置简单

搭建过程

整个博客的搭建过程非常顺畅。Astro 的官方文档写得相当好，按照指南一步步操作即可。

# 创建新项目
npm create astro@latest my-blog
cd my-blog
npm install
npm run dev

开发服务器启动后，打开 http://localhost:4321 就能看到默认页面。接下来就是根据自己的喜好定制主题。

极简文艺风设计

这个博客的设计灵感来自 Medium 和 Readwise Reader。核心设计原则是：

  1. 大量留白，让内容呼吸
  2. 使用衬线字体（Lora）作为标题字体
  3. 极简的配色方案，以黑白灰为主
  4. 精心设计的排版，确保长文阅读体验
好的设计是看不见的设计。读者应该专注于内容，而不是页面本身。

在线写文章

通过集成 Decap CMS（原 Netlify CMS），可以在线编写和发布文章，无需每次都通过 Git 提交。只需访问 /admin 页面，登录 GitHub 账号即可。

总结

用 Astro 搭建博客的体验非常愉快。如果你也在寻找一个轻量、快速、专注于内容的博客方案，强烈推荐试试 Astro。
